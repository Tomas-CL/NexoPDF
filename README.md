# Nexo PDF

Voy a hacer una app en **Python** con una interfaz gráfica utilizando la librería **Tkinter** que escanee toda la carpeta del usuario (incluyendo subcarpetas) para buscar todos los archivos con la extensión `.pdf`y copiarlos en una carpeta aparte, evitando duplicaciones.

Si un archivo ya exite la carpeta de destino no se copia; solo se copian nuevos archivos `.pdf` nuevos.

Una vez que la app termine de escanearlos y copiarlos debe abrir automaticamente la carpeta de destino para ver que el usuario pueda ver todos los archivos `.pdf` copiados.
